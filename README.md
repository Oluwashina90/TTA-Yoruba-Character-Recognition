# Test‑Time Adaptation for Low‑Resource Handwritten Yoruba Character Recognition

This repository contains the code, data, and results for the paper:

> *Test‑Time Adaptation for Low‑Resource Handwritten Character Recognition: A Study on YorubaNet and ResNet‑18 under Synthetic Distribution Shifts*  
> Oluwashina Oyeniran (2026)

## Abstract

We investigate whether a lightweight test‑time adaptation (TTA) method – updating batch normalisation statistics – can improve robustness of handwritten character recognition for the Yoruba script under synthetic corruptions (blur, salt‑pepper noise, thinning). Using a dataset of 12,600 images (35 classes), we compare a custom CNN (YorubaNet) with ResNet‑18. TTA significantly improves accuracy for noise and thinning (up to +25% for YorubaNet) but fails for blur. Full statistical analysis and publication‑ready plots are provided.
